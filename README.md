Altium Designer 26.3.0 Build 6 – Description
Release Date: February 5, 2026

Altium Designer 26.3.0 Build 6 is a routine update in the Altium Designer 26 series, delivering several targeted improvements across PCB design, harness design, platform functionality, and data management. Below is a summary of the key changes in this release.

PCB Design Improvements
Enhanced SOLIDWORKS and Parasolid Support

Added support for SOLIDWORKS 2024 and 2025 models (*.SldPrt) when working with 3D bodies.

PCB export to Parasolid file format (*.x_t) now uses Parasolid version 35.1, enabling newer SOLIDWORKS versions (2024 and 2025) to correctly open and import exported files.

Bug Fixes

Fixed an issue where certain component 3D models failed to convert when exporting specific PCBs to Parasolid format, causing errors in SOLIDWORKS.

Resolved an error when placing a layer stackup table in the design space with certain Advanced Settings enabled.

Fixed an error when using interactive routing with Loop Removal functionality.

Corrected an issue where Min/Max Size constraint changes for Fabrication/Assembly Testpoint Style rules were not being saved.

Addressed multiple Access Violation errors in various PCB editing scenarios.

Harness Design Improvements
Jumper Wire Display

Jumper wires (connecting two cavities of the same connector) defined in a wiring diagram are now correctly displayed in the associated layout drawing.

When a bundle is selected, the Properties panel’s Bundle Objects region includes these jumper wires as part of the bundle.

Such wires support only manual length definition; entered values are included in ActiveBOM and manufacturing drawings (BOM table and wiring list).

Improved “Update From Libraries” Feature

When accessed from a wiring diagram, the update now includes wires, cavity components, and associated components.

When accessed from a layout drawing, the update now includes coverings, labels, and associated components.

Note: For objects to be included, their assigned component must contain a link to a schematic symbol.

Platform Improvements
Enhanced notifications in Altium Designer Develop for subscription expiration scenarios (approaching expiration, expired with read-only workspace, and no available Author seats).

The End User License Agreement (EULA) has been replaced with the Master Services Agreement (MSA) when installing Altium Designer Develop and Altium Designer Agile. Altium Designer continues to use EULA (which has also been updated).

Data Management Fixes
Manufacturer Part Search panel now correctly displays filter names and parameters; scrolling issues in the Details pane have been fixed.

Fixed an issue where component revision states were not updating after refreshing the Components panel.

Resolved an error when placing reuse blocks from the Explorer panel.

Fixed an issue where usernames appeared as “Anonymous” when viewing comments in projects.

Corrected an error when attempting to release changes to existing components in Single Component Editing mode.

Import/Export Fix
Fixed an issue where capacitor values specified in fF (femtofarads) were not recognized when exporting PCB to Ansys EDB format, incorrectly defaulting to 1nF.

Multi-Board Design Fix
Resolved a critical issue where Altium Designer would completely freeze when attempting to map pins between inline connectors in a harness within a multi-board schematic.

This update focuses on stability improvements, enhanced 3D mechanical collaboration capabilities, and refined harness design workflows. Users working with SOLIDWORKS integration or complex wire harness designs will benefit most from this release.
